URI that will be linked with every File added to a Rally Changeset's Change.

If you have a web-accessible repository browser and wish to deeply link into that system with files linked in Rally Changes, use the ${repository}, ${revision} and ${file} variables to do so.

For example, if you want a file with name README.md and commit hash 12ab34cd56ef to be accessible at http://github.com/your-name/your-project/blob/12ab34cd56ef/README.md then you would set this property as: http://github.com/your-name/your-project/blob/${revision}/${file}

Or if you defined your rallyScmName to be your-name/your-project, then http://github.com/your-name/your-project/blob/12ab34cd56ef/README.md then you would set this property as: http://github.com/${repository}/blob/${revision}/${file}
